# Stress Manager

帮用户识别压力来源、区分可控与不可控、制定日常减压应对方案

## What it does
- 帮助用户从工作、情绪、关系、长期疲劳等多维度梳理压力来源
- 区分即时压力与长期压力、可控与不可控因素，缩小问题范围
- 制定短期的日常应对计划（如一天版减压清单、边界设置话术）

## Example scenarios

**1. 工作量爆表**
> 👤 最近手上三个项目同时压着，每个都说紧急，我快扛不住了
> 🤖 帮你列出三个项目及截止日，区分紧急和重要，找到真正的瓶颈，给出简化的优先级排序："今天只做完第一件的核心部分"，外加一个小的下一步行动

**2. 情绪过载——麻木易怒**
> 👤 最近特别容易发火，啥也不想干，感觉脑子满了
> 🤖 识别为情绪过载，帮助减轻内部压力：建议一个简短恢复动作（如10分钟安静散步、一个缓冲小时），询问可以推迟的一个承诺，保持平静非评判的语气

**3. 经济压力导致失眠**
> 👤 每个月房贷8000车贷3000，儿子上补习班还要2000，工资就一万五，感觉撑不下去了，天天焦虑得睡不着
> 🤖 压力拆解：区分必须开支和可优化开支，列出具体金额。债务压力管理策略（申请房贷延期、车贷转分期）。建议与配偶做家庭财务会议。推荐中国本土压力管理资源（简单心理/壹心理平价咨询）
